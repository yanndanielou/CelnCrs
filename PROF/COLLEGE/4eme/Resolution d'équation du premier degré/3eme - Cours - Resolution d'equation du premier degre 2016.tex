\section{Généralités sur les équations}
 
 
\df{
Une équation est une égalité dans laquelle figure un nombre inconnu, désigné en général par une lettre qui est appelée l'inconnue.
}\\

\underline{\textbf{Exemple} }:\\


$2x - 11 = 7 - x$ est une équation dans laquelle l'inconnue est désignée par $x$. \\

\begin{center}
$\underbrace{2x-11}_{\text{Premier membre}} = \underbrace{7-x}_{\text{Second membre}}$\\
\end{center}

\textbf{\q Le nombre 3 est-il solution de l'équation $2x - 11 = 7 - x$ ?}

\bmul{2}
D'une part, $2 \times 3 -11 = \underline{- 5}$

\columnbreak

D'autre part, $7- 3 = \underline{4}$
\emul

L'égalité n'est donc pas vérifiée pour $x=3$.\\

\underline{Conclusion }: Le nombre 3 n'est pas solution de l'équation.\\

\textbf{\q Le nombre 6 est-il solution de l'équation $2x - 11 = 7 - x$ ?}

\bmul{2}
D'une part, $2 \times 6 -11 = \underline{1}$

\columnbreak

D'autre part, $7- 6 = \underline{1}$
\emul

L'égalité est donc vérifiée pour $x=6$\\

\underline{Conclusion }: Le nombre 6 est une solution de l'équation.\\

\vspace*{0.5cm}

\df{
Résoudre une équation, c'est trouver toutes les solutions qui vérifient cette équation.
}\\


\begin{Exoapp}

\reponse[2]\\

\reponse[2]\\

\end{Exoapp}




\section{Résolution d'équation}
 
\subsection{Équation du type a + x = b}

\vspace*{0.5cm}

$\rightarrow$ \textbf{Résoudre l'équation $3+x=7$.}\\



\begin{center}
$3 + x = 7$\\
$\cancel{3} + x \textcolor{red}{- \cancel{ 3}} = 7 \textcolor{red}{- 3}$\\
$x = 7 - 3$\\
\fbox{$x = 4$}

\end{center}


La solution de l'équation $3 + x = 7$ est le nombre 4. On notera alors $\mathscr{S}$ = $\left\{4\right\}$.\\

\subsection{Équation du type ax = b}

\vspace*{0.5cm}

$\rightarrow$ \textbf{Résoudre l'équation $-5x = 125$}.\\





\begin{center}
$$-5x = 125$$

$$\dfrac{-5x}{\textcolor{red}{{-5}}} = \dfrac{125}{\textcolor{red}{-5}}$$

$$\dfrac{\cancel{-5}x}{\cancel{-5}} = \dfrac{125}{-5}$$
$$x = - \dfrac{125}{5}$$
\fbox{$$ x = -25$$}
\end{center}


La solution de l'équation $-5x = 125$ est le nombre - 25. On notera alors $\mathscr{S}$ = $\left\{-25\right\}$.\\

\subsection{Équation avec des inconnues dans les 2 membres }

\vspace*{0.3cm}

$\rightarrow$ \textbf{Résoudre l'équation $7x - 2 = 6 + 5x$}.\\

\bmul{2}

$$7x - 2 = 6 + 5x$$

$$7x - 2 - 5x = 6 + 5x - 5x$$

$$2x - 2 = 6$$

$$2x - 2 + 2 = 6 + 2$$

$$2x = 8$$

$$\fbox{x=4}$$

$\mathscr{S}$ = $\left\{4\right\}$.\\



\columnbreak

\begin{center}
\textbf{\underline{Méthode de résolution }}\\
\end{center}

On commence par isoler l'inconnue dans un des
deux membres.\\

On regroupe ensuite les nombres sans $x$ dans l'autre
membre.\\

On utilise alors les méthodes de résolution vues juste avant.\\

On n'oublie pas de donner la ou les solution(s).\\

\emul
