\underline{\textbf{Rappels de cours}}\\

\textbf{Définition :} Une expression littérale est une expression dans laquelle un ou plusieurs nombres sont représentés par des lettres.\\

\textbf{Propriété :} Le signe "$\times$" peut être supprimé :
\bi \item devant une lettre ;
\item devant une parenthèse.
\ei

\underline{Exemples :} $3 \times a$ s'écrit 3a   \hspace*{0.5cm}  $b \times 4 \times c$ s'écrit 4bc\\

\textbf{Définition :} Soit a un nombre quelconque.
\bi \item	$a \times a$ se note $a^{2}$
\item	$a \times a \times a$ se note $a^{3}$  

\ei

\vspace*{0.2cm}
\end{minipage}}
\end{flushleft}
