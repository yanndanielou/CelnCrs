\subsection{Les évènements}




\df{
On appelle \textbf{événement} une condition qui peut ou non être réalisée lors d'une expérience.
}\\

\vspace*{0.3cm}

\underline{\textbf{Exemple} }: Considérons une expérience aléatoire où on lance un dé bien équilibré et on note le chiffre sur la face du dessus.\\

\vspace*{0.5cm}

\df{ 
\bi
\item 	Un évènement qui ne peut être réalisé que par une seule issue est appelé un évènement élémentaire.\\

 \item Un évènement qui a une probabilité égale à 1 est un évènement certain.\\
 
\item Un évènement qui a une probabilité égale à 0 est un évènement impossible.\\
\ei
}\\
